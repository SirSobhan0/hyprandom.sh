# hyprandom

## Usage: 
```
hyprandom.sh [-s <source>] [-u] [-q]
Options:
  -s <source>   Specify wallpaper source.
  -u            Use unsafe mode for sources.
  -q            Use custom query for some sources/url for fetching from url.
  -h            Display this help message.
```
