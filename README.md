# hyprandom

## Usage: 
```
hyprandomV2.sh [-s <source>] [-u] [-m] [-q]
Options:
  -s <source>   Specify wallpaper source.
  -u            Use unsafe mode for sources.
  -q            Use custom query for wallhaven.
  -h            Display this help message.
```
